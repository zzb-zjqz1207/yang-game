# 羊羊消消乐 - 类似"羊了个羊"的消除游戏

## 🎮 游戏说明

一个简单有趣的堆叠消除小游戏，玩法类似"羊了个羊"。

## 📖 玩法规则

1. 点击卡片，卡片会移动到下方槽位
2. 槽位中凑齐 3 张相同的卡片即可消除
3. 槽位满 7 张则游戏结束
4. 消除所有卡片即可通关！

## 🚀 如何运行

直接在浏览器中打开 `index.html` 即可开始游戏。
